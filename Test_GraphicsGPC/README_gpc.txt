To make:

1. Get hold of the General Polygon Clipper.
http://www.cs.man.ac.uk/~toby/alan/software/

2. Copy gpc.c and gpc.h into the camlgpc directory
3. Type "make" to build
4. Type "make documents" to build the documentation in doc/camlgpc/html
5. Type "make -f examplemake" to build the example.
6. Type "./test" to run the test.
